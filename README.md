# MyTasks

A simple and minimal to-do list application for Android.

## Features

- **Add Task** — Type a task in the input field and tap the add button or press enter to create a new task.
- **Edit Task** — Tap the edit icon on any task to modify its text via a dialog.
- **Delete Task** — Tap the delete icon on any task to remove it from the list.
- **Mark Complete** — Tap the checkbox next to a task to mark it as done. Completed tasks appear with a strikethrough.
- **Drag to Reorder** — Long-press the drag handle on the left side of any task and drag it up or down to reorder manually.
- **Local Storage** — All tasks are stored locally on the device using SQLite. Data persists across app restarts.
- **Works on Android 5+** — Minimum SDK version is API 21 (Android 5.0 Lollipop), covering nearly all active Android devices.
